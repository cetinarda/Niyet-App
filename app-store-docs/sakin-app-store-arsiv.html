5. App Review Notes (Tam Metin)

Bu metni App Store Connect > App Review Information > Notes alanına yapıştır.

**Sakin** is a free-to-download **Lifestyle** application with in-app purchases that guides users through a daily mindfulness routine — from morning intention to evening closure. The app does not make any medical claims; it focuses on "reminders", "habit building" and "personal awareness".

**Pricing:** Free download with two premium plans:
- **Yearly Subscription:** $4.99/year (Auto-Renewable) — Product ID: app.sakin.life.yearly
- **Lifetime License:** $19.99 (Non-Consumable) — Product ID: app.sakin.life.lifetime

## Technical Notes for Review Team

### Account / Demo
- The app does not require account registration or login.
- No demo account needed.
- The app works immediately upon launch.

### Main Flow
1. Launch screen → Tap "Hazırım" (I'm ready)
2. Write a morning intention (any text)
3. Select 3 motivation words
4. Breathing exercise screen (tap a mode, then Start)
5. Sound Waves — tap a solfeggio frequency to listen (accompanied by bird sounds)
6. 22 Chakra Awareness — view energy center info, start 60-second connection
7. Daily reminders checklist
8. Evening closure — write reflection notes
9. Connection screen — human body chakra system with progress tracking
10. Weekly AI-generated inner report

## Category and Content Compliance

| Topic | Detail |
|---|---|
| Category | Lifestyle — no medical/health claims |
| Secondary Category | Entertainment |
| HealthKit | Not used |
| Medical claims | None — uses "awareness" and "reminder" language |
| Spiritual content | Chakra terms used in cultural/traditional context; does not represent any specific religion |
| AI features | Uses Groq API (Meta Llama) for personalized text analysis. Explicit user consent required before first use |
| User data | Stored locally on device only. AI features send anonymous text to API — no personal identifiers |
| In-App Purchases | $4.99/year subscription + $19.99 lifetime (via StoreKit / cordova-plugin-purchase) |
| External Payments | None in iOS app. Web version uses separate payment provider |
| Audio | Solfeggio frequency tones (Web Audio API) and bird sound recordings from local assets. No streaming |

## AI-Powered Features
- Inner Mirror: Analyzes user-written text, provides personalized awareness reflection
- Emotional Awareness Analysis: Maps emotional patterns to body awareness
- Weekly Report: Generates weekly summary based on journal entries and frequency listening time
- AI provider: Groq API (Meta Llama 3.3 70B)
- Groq does not use submitted data for model training
- All communication over HTTPS
- AI responses include disclaimer about not being medical advice

11. Apple Ret Gerekçeleri & Çözümler

Guideline 1.4.1 — Safety: Physical Harm

Sorun: Uygulama içinde tıbbi teşhis, tedavi ve şifa iddiaları vardı.

Yapılan Düzeltmeler:

  • Tüm organ bağlantıları kaldırıldı (böbrek üstü bezleri, timus, tiroid, epifiz vb.)
  • "DNA onarımı", "Hücresel uyum", "Travma temizliği" ifadeleri kaldırıldı
  • Louise Hay hastalık-duygu eşleştirmesi → "Duygusal Farkındalık Rehberi"
  • Reiki tedavi protokolleri → "Enerji Farkındalık Rehberi"
  • Şifalı bitki tavsiyeleri kaldırıldı
  • AI promptlarına "Asla tıbbi teşhis koyma" uyarısı eklendi
  • "Mucize Tonu", "aktive eder" ifadeleri kaldırıldı
  • "aktive olduğunda" → "deneyimlendiğinde"
  • "enerji şifa yöntemi" → "farkındalık pratiği"
  • Nefes açıklamalarında "parasempatik sistemi aktive eder" kaldırıldı
  • Tıbbi uyarı metni tüm AI yanıtlarına eklendi
  • Frekans ve çakra içeriklerine kaynak referansları (Gregoryen ilahileri, Veda geleneği) eklendi

Guideline 2.3.8 — Accurate Metadata

Sorun: Metadata uyuşmazlığı, yer tutucu ikon, harici ödeme.

Yapılan Düzeltmeler:

  • LemonSqueezy iOS sürümünden kaldırıldı (sadece web'de)
  • Kategori: Health & Fitness → Lifestyle + Entertainment
  • Keywords'ten "reiki" ve "wellness" kaldırıldı
  • "Çakra Yükselişi" → "Çakra Farkındalığı"
  • Yeni zen-minimalist app icon tasarlandı (yer tutucu değil)
  • Fiyat modeli güncellendi: Ücretsiz + IAP
  • iOS'ta StoreKit ile IAP entegrasyonu yapıldı
